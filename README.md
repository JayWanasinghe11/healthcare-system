# Healthcare and Inventory Management System

This is a professional full-stack healthcare application designed to manage clinical operations, pharmacy inventory, and administrative analytics in one unified platform.

## Key Features
- **Patient Registration**: Secure management of patient demographics (NIC, DOB, etc.).
- **Inventory Control**: Real-time stock updates for pharmacy medicine using database triggers.
- **Business Intelligence**: Analytics dashboards for medicine usage trends and patient registration forecasting.
- **Audit Logging**: Enterprise-grade security tracking for all system activities.

## Technology Stack
- **Frontend**: React.js, Tailwind CSS (Glassmorphism design)
- **Backend**: Node.js, Express REST API
- **Database**: Oracle SQL (with PL/SQL stored procedures and triggers)

## Getting Started
1. **Clone the repository:**
```bash
   git clone <https://github.com/JayWanasinghe11/healthcare-system.git>
2. Run `npm install` to install dependencies.
3. Configure your Oracle DB in the `.env` file.
4. Run `npm start` to launch the application.
